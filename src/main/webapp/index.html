Junit:
  • Open source framework designed for writing and running tests in the Java programming language.
  • This framework works on testing one block at a time rather than module level functional testing.


    • Benefits:
    1. Helps to identify how much code implementation deviates from the goal.
    2. Helps to code and test both during the development.
    3. Helps to make sure that the modifications in the code will not break the system.
    EclEmma:
  • Java code coverage tool which brings code coverage analysis directly into the Eclipse workbench.
  • Helps identify what percentage of your application is tested and what is not.


    • Analysis:
    It consists of two parts:

  • Coverage View:
    1. Coverage ratio
    2. Items covered.
    3. Items not covered.
    4. Total items.
  • Source Highlighting:
    1. Green for fully covered lines,
    2. Yellow for partly covered lines.
    3. Red for lines that have not been executed at all.
    • Selenium:
    Automating web applications for testing purposes. Helps in load testing. Helps testing the UI of the application.
